windows的任务计划程序（可以设定什么时间启动程序，比如每天或者每次重启后）


1. 打开任务计划程序
按 Win + R 打开运行对话框，输入 taskschd.msc 并回车，这将打开任务计划程序。
2. 创建新任务
在任务计划程序中，选择“操作”菜单下的“创建基本任务...”。
3. 设置任务
名称和描述：给你的任务起一个名字，如“AutoLoginScript”
触发器：在“触发器”选项中，选择“当我登录时”或“当计算机启动时”
操作：选择“启动程序”。
程序/脚本：点击“浏览”，找到并选择你的Python解释器的可执行文件。通常情况下，它位于如C:\Python39\python.exe的路径（取决于Python版本和安装位置）。
添加参数（可选）：在“添加参数（可选）”区域，输入你的脚本的路径，如C:\path\to\auto_login.py。
